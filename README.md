## ⚙️ Installation & Setup

Follow these steps to run this project locally:

**1. Clone the repository**
```bash
git clone https://github.com/YOUR_GITHUB/your-repo-name.git
cd your-repo-name
```

**2. Install frontend dependencies**
```bash
cd frontend
npm install
```

**3. Install backend dependencies**
```bash
cd backend
pip install -r requirements.txt
```

**4. Configure environment variables**
```bash
cp .env.example .env
# Fill in your API keys and database credentials
```

**5. Run the development servers**
```bash
# Frontend
npm run dev

# Backend (in a separate terminal)
python manage.py runserver
```

> The app will be live at `http://localhost:5173` (frontend) and `http://localhost:8000` (backend).

## 📖 Usage

Once both servers are running:

1. **Open** `http://localhost:5173` in your browser
2. **Register** or **log in** to your account
3. **Explore** the features based on your role (Farmer / Buyer / Admin)
4. **Interact** with the marketplace or attendance system in real time
